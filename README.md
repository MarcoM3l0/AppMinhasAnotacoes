# AppMinhasAnotacoes
Prova de proficiência da matéria de desenvolvimento mobile: O aplicativo "Minhas Anotações" permite que os usuários criem, visualizem, editem e excluam notas rápidas no dispositivo móvel. O objetivo é proporcionar uma ferramenta simples para o registro de notas pessoais, ou seja, informações, ideias ou lembretes importantes de forma organizada.
